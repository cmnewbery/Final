Iceland Is Awesome!

Iceland, a Nordic island nation, is defined by its dramatic landscape with volcanoes, geysers, hot springs and lava fields. Massive glaciers are protected in Vatnajökull and Snæfellsjökull national parks. Most of the population lives in the capital, Reykjavik, which runs on geothermal power and is home to the National and Saga museums, tracing Iceland’s Viking history.

Below are some classes I am taking this semester.

IT 315: Website and Webpage Design
IT 315 is great web design course. Taking this course will teach you the basics of setting up a website using HTML, Cascading Style Sheets (or CSS), and Java Script. Not only is web design a great tool to have but it is a fun hobby and is personally my favorite class this semester.

IT 317: Enterprise Information Architecture
IT 317, or Enterprise Information Architecture, is a course that teaches basic computer skills and provides information on how a computer works. This is also one of my favorite classes and we meet two times a week just before noon. We are currently learning processor technology and implementation but have gone over computer architecture, data conversions, and data representations.

ECON 301: Managerial Economics
I attend Professor Rahoui's Economics 301 class once a week. This class is a managerial economic class. I have taken both Micro and Macro Economics with Professor Rahoui. This semester we are learning about key topics like demand analysis, economic forecasting, production, and cost analysis.

MGMT 325: Contemporary Organizations and Management
Contemporary Organization and Management teaches important skills for a manager in today's workplace. The course focuses on planning, organizing, and leading. I attend this class for three hours, once a week.

FIN 220: Wallstreet 101
Wallstreet 101 is a very interesting class. I was interested in this class because I have invested a small amount of money is a few publicly traded companies. Students work with the Bloomberg program and terminals to practice basic stock knowledge and experience. Hopefully I will understand how to buy and sell stocks more efficiently.
